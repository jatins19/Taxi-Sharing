# Taxi Sharing Platform

A modern taxi booking platform connecting passengers with nearby drivers while providing real-time ride tracking and secure online payments.

## Features

- User & Driver Registration
- Ride Booking
- Live Tracking
- Fare Estimation
- Payment Gateway
- Ride History
- Driver Dashboard
- Admin Dashboard
- Ratings & Reviews
- Push Notifications

## Tech Stack

Flutter

React

Node.js

Google Maps API

Firebase

MySQL

## Installation

```bash
git clone <repo>

npm install

npm run dev
```
